Capabilities

Project Workspace Engine ▼
Create and switch between multiple projects. The dashboard reads your project folders and lets you pick one. Everything you do—editing, deploying, running—happens inside the selected project.
File System Access ▼
Use the FileTree panel to browse your project. Click any file to open it in the inline editor. Save changes instantly. For deep work, open the full Editor page.
Inline Dashboard Editor ▼
When you click a file in the FileTree, it opens in the right column. Edit it directly and hit Save. This is perfect for quick fixes and small edits.
Full Editor Page ▼
Go to /editor for a full-screen editing environment. Load any file by typing its path. Save instantly. This is your deep-work zone.
Cloudflare Suite ▼
Deploy Workers, Pages, D1, R2, KV, and Queues. Use the quick panel for fast actions or open the full Cloudflare page for detailed control. Logs appear instantly so you can see what happened.
GitHub Integration ▼
Clone any GitHub repo into a project. The backend handles the clone and logs the events. After cloning, the project appears in your Project Switcher.
Run / Deploy Engine ▼
Start backend or frontend processes for your project. Use the quick panel for instant runs. Logs appear in the main view so you can see what’s happening.
Plugin System ▼
Add custom tools, modules, and UI panels. Plugins can extend the dashboard, add pages, or add backend logic. This is how the Playground grows with you.
Preview Panel ▼
The iframe on the dashboard can load any local app or generated output. Perfect for testing UI, static sites, or generated pages.
Logs & Diagnostics ▼
View system logs, deployment logs, and builder events. The Logs page shows everything the backend reports.
Theme Engine ▼
Switch between Industrial and Neon Hacker themes instantly. The entire UI updates without reloading the page.
