MCU Lab — Capabilities

A complete overview of what the Boardwalk MCU Lab Pro can do.

Supported Boards

  • ESP32 DevKit V1 Default
  • ESP32‑S3
  • ESP8266 NodeMCU
  • ESP32‑C3 (via backend registry)
  • ESP32 WROOM / WROVER modules

Boards are loaded dynamically from backend/microcontroller/boards.py.

Wiring Engine

  • Drag‑and‑drop components
  • Auto‑ID assignment for components
  • Wire mode with color cycling
  • SVG‑based wire rendering
  • Zoomable breadboard workspace
  • Auto‑generated netlist JSON

Component Library

Over 40 components available:

  • LEDs, resistors, buttons, potentiometers
  • Environmental sensors (DHT11, DHT22, BME280, MQ‑series)
  • Motion & distance sensors (PIR, HC‑SR04, VL53L0X)
  • Displays (OLED SSD1306, LCD1602, TFT 1.8")
  • Communication modules (NRF24L01, LoRa SX1278, HC‑05, ESP‑01)
  • Power modules (buck, boost, solar, battery holders)

AI Panel Pro

  • Wiring mode — JSON wiring plans, pin maps, diagrams
  • Code mode — deterministic firmware generation
  • Image mode — blueprint, photorealistic, anime, technical
  • Session memory for multi‑step reasoning
  • Project‑optional AI (works with or without a project)

Simulation Engine

  • Rule‑based electrical validation
  • LED polarity + resistor checks
  • Over‑voltage detection
  • Over‑current detection
  • Short‑circuit detection
  • Sensor voltage compatibility checks

Powered by backend/microcontroller/breadboard_sim.py.

Firmware Generation

  • Natural‑language firmware generation
  • ESP32 / ESP8266 C++ (Arduino‑CLI)
  • Auto‑scaffolded project structure
  • Board‑aware code generation
  • Merge strategies (overwrite / append)

Flashing Engine

  • Arduino‑CLI compile + upload
  • Auto‑detect serial ports
  • Board‑specific FQBN selection
  • Optional erase mode
  • Structured logs for compile + upload

Import / Export

  • Import firmware project → auto wiring
  • Export netlist JSON
  • Load saved netlists

Playground Integration

  • Works inside the main Playground Studio
  • Uses the same backend (FastAPI)
  • Shares project manager + file tree
  • Shares AI assistant + terminal
